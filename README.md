# 📊 Sales Data Analysis with Python

## 🧠 Project Overview
This project analyzes sales data using Python and Pandas to extract key business insights.

## 📊 Objectives
- Understand total sales performance  
- Identify top-performing cities  
- Determine best-selling products  

## 🛠 Tools & Technologies
- Python  
- Pandas  
- Matplotlib

## 🚀 How to Run
pip install -r requirements.txt
python analisis.py

## 💼 Business Value
This analysis helps companies focus on high-performing markets and optimize product strategies.
